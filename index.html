News Polarizer

Technology. This app uses the MERN framework to explore sentiment in news articles. It uses RapidAPI to fetch news articles based on a search query. Twelve of these articles are submitted to the IBM Cloud NLU for analysis. Headlines for the three articles with the highest, lowest, and the median scores are displayed, with links to the original source. Each set of articles are visualized on a dot graph.

Google Books Search

Technology. This app uses the MERN framework (MongoDB, Express, React, and NodeJS). It utilizes the Google Books API to search for books in its extensive database. The user has the option to save returned books to MongoDB, and later delete them. Dependencies include Axios and Mongoose.

Vacay

Technology. This app is a group project. It provides a snapshot of information that may be useful for travelers. There are API's to obtain country data, Google Maps, weather information and a currency converter. An itinerary table allows the user to store and edit their trip plan. Itinerary data is stored in Firebase. My role was to setup the API's, Firebase, design the layout, and handle code review for my team.

Web News Scraper

Technology. This NodeJS Express app extracts world headlines from the Jerusalem Post. The view engine is Express-Handlebars. Headlines and related metadata are extracted with Axios and Cheerio and saved to MongoDB using Mongoose. Users can view the original article, save it for later viewing, or remove it, and add their own notes in a modal bootbox, also stored in MongoDB.

Clicky App

Technology. This React app tests the user's memory. React renders 12 images that are reshuffled with each click. There are four component folders in this SPA that tracks the user's score. CSS effects applied are a hover class that enlarges the image and a shaker animation identifying a wrong selection.

Burger Sequel

Technology. An Express Server app using Sequelize to connect with a MySQL database. Express-Handlebars is the view engine. A Handlebars Helper function was created to list burgers devoured by each customer when rendered.

Friend Finder

Technology. This NodeJS app has an array of "friend stats" to compare against the results of a survey questionnaire. This was created around Halloween so expect some unusual matchups. Users can add their own personal image.

Bamazon

Technology. A simple NodeJS CLI purchasing solution serving customer, manager, and supervisor roles. NPM modules used are Inquirer, MySQL, and Console.table.
